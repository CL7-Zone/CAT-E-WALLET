## Những chức năng chính - Main functions

```js
Đăng nhập - Login 
Đăng xuất - Logout
Đăng ký - Sign-up
Xác minh email khi đăng ký tài khoản - verify email when account sign-up
Quét hình ảnh hóa đơn nạp tiền của khách hàng để xác minh - Scan invoice image to verify
Tăng số dư trong tài khoản cuả khách hàng - Increase the amount in account of customers
```

## Mô tả dự án - Description project

```js
- Ứng dụng ví tiền điện tử giúp khách hàng giao dịch, mua 
bán hàng trực tuyến trên các nền tảng thương mại điện tử.
Ứng dụng có tích hợp module quét hình ảnh để xác thực hóa
đơn nạp tiền của khách hàng 1 cách chính xác nhất và an toàn.

END SUB (A digital wallet application helps customers conduct transactions 
and shop online on e-commerce platforms. The application integrates 
an image scanning module to accurately and securely verify 
customers' deposit receipts.)

```
