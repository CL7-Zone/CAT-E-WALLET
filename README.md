# TÊN DỰ ÁN CAT E-WALLET PROJECT


## Những chức năng chính - Main functions

```js
Đăng nhập - Login 
Đăng xuất - Logout
Đăng ký - Sign-up
Xác minh email khi đăng ký tài khoản - verify email when account sign-up
Quét hình ảnh hóa đơn nạp tiền của khách hàng để xác minh - Scan invoice image to verify
Tăng số dư trong tài khoản cuả khách hàng - Increase the amount in account of customers
```

## Mô tả dự án - Description project
